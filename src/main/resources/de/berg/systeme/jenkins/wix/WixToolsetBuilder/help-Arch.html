The platform supported by the package. This option works on compiler candle.exe.

This attribute's value must be one of the following:

  • x86Set this value to declare that the package is an x86 package. The package will also run on x64 and ia64.
  • ia64Set this value to declare that the package is an ia64 package. The package will not run on x86 or x64.
  • x64Set this value to declare that the package is an x64 package. The package will not run on x86 or ia64.
